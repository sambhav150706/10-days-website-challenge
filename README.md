# Personal Portfolio (HTML/CSS/JS)

## How to run
- Open `index.html` in your browser.

## Customize
- **Name & tagline**: edit the Hero section in `index.html`
- **About**: update text in `#about`
- **Skills**: update cards in `#skills`
- **Projects**: replace the placeholder project cards in `#projects`
- **Contact**: update your email/social links in `#contact`

## Notes
- The contact form includes client-side validation and a success message, but **does not send emails** (no backend).
